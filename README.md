# ParaCL(Two Forces) language

## Installing project:
1. Clone this repository to your computer:
```
    git clone git@github.com:Ignitor21/TwoForces.git
    cd TwoForces
```
2. Run cmake:
```
    cmake -B build
    cd build 
    cmake --build .
```

You can find source code examples in folder `examples`
